== Description ==

### The best Marketo plugin for WordPress.
Easily add contacts into Marketo when users submit a Gravity Forms form.

Map your Gravity Forms form fields to Marketo data, and have contacts automatically updated if the contact already exists.

####Easily view contacts in Marketo
When the Entry is created, a link to the Contact's page in Marketo is shown inside WordPress.

Requires a server with PHP 5+ and SOAP installed.

== Screenshots ==

3. Then you map the fields in the form to the fields for the Marketo contact.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
1. Go to the plugin settings page (under Forms > Settings > Marketo)
1. Enter the information requested by the plugin.
1. Click Save Settings.
1. If the settings are correct, it will say so.
1. Follow on-screen instructions for integrating with Marketo.
